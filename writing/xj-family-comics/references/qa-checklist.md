# QA Checklist（漫画版）

## 必过项

- 任务画像（封面）与每格均为 4:3 横版单图。
- **手绘漫画感**：可见墨水笔触、略带抖动的草图线，不是平滑矢量 / 干净贴纸风；背景为带轻微纸感的暖色调。
- 三口之家都按 `characters.md` 固定角色块出现且形象一致：
  - 饺子：大脑袋、大眼睛双眼皮、1 岁婴儿感、无语言、不学龄化。
  - 小鞋子：3 个发髻（核心识别）、大眼睛、知性温柔、**不拿鞋**。
  - 小修：圆框眼镜 + 邋遢（乱发 / 胡茬 / 歪领）、大脑袋、眼神清醒。
- 画风统一：手绘漫画、柔色、圆润可爱但不幼稚恶心，不是表情包 / 日漫 / 写实。
- 每格只讲一个核心动作 / 情绪瞬间；任务画像为全家福封面，无台词、无标题文字。
- 台词气泡少、短、可读；饺子用表情气泡（咿呀 / ？ / ！）。
- 幽默与哲理都在：笑点成立，结尾有积极思考落点，不油腻、不苦情、不空洞说教。
- 无「图片由 AI 生成」类水印残留（生成提示已要求不加水印，仍要复检）。

## 失败信号（出现则重生成或局部编辑）

- 妈妈发髻数量变了（不是 3 个）或披发。
- 妈妈**拿了鞋**（已撤销该设定，任何格/画像里都不应出现大鞋）。
- 小修没眼镜 / 太干净不邋遢 / 变成纯丑角无哲理。
- 饺子被画成会说话的学龄儿童或比例不像 1 岁。
- 跨格角色长相 / 服装 / 画风跳变。
- 画面太干净像矢量贴纸、或太暗、太写实、太像日漫战斗 / 表情包。
- 哲理过重变说教，或笑点太油变段子。
- 任务画像出现标题文字 / 台词气泡（封面应纯净）。
- 中文台词错字严重、气泡不可读。
- 背景重噪点、脏污做旧、纸纹过重。
- 和某篇已有构图 / 笑点过于雷同。

## 迭代方法

- 角色跳变：用 `prompt-template.md` 的「增强一致性」提示重生成该格，强调三口之家固定特征（妈妈 3 髻、不拿鞋）。
- 太干净像贴纸：强调 hand-drawn comic, visible ink pen strokes, sketchy wobbly lines, paper grain。
- 太暗 / 太写实：强调 warm hand-drawn comic, soft pastel, rounded, not realistic。
- 太像表情包：强调 not sticker / not meme，收敛夸张表情。
- 太说教：把哲理后移、交给饺子动作或小修歪理带出，减少妈妈旁白式台词。
- 文字错：优先局部编辑；错得多就重生成并减少气泡字数。
- 笑点弱：回到 `outline-method.md` 的幽默来源，加反差 / 翻车 / 小修歪理。

## 交付判断

高质量一格应当：第一眼温暖手绘、想笑；细看发现「哦，这说的有点道理」。任务画像应让人一眼认出这家三口与当期主题，干净无字。
