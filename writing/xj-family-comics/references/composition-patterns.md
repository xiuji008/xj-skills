# 分格构图与分镜

每格是独立的 4:3 横版单图。本文件讲单格怎么画、怎么分配角色与台词气泡。

## 单格三要素

1. **主体动作**：这一格「正在发生什么」。只画一个核心动作，不要塞两个冲突动作。
2. **角色站位**：谁在画面里、靠哪边、占多大。默认饺子视觉最亮最靠前；小鞋子偏稳重居中或一侧；小修偏一侧、姿态松垮。
3. **台词气泡**：预留右上 / 左上 / 角色头顶空间放气泡。饺子用「咿呀 / ？ / ！」类表情气泡，小鞋子与小修用短对话气泡。

## 角色同框站位（常用）

- 三人并排坐沙发 / 趴地陪玩：饺子居中（最萌），爸妈分坐两侧。
- 爸妈逗饺子：一人抱 / 扶，一人对视。
- 小修单独歪理：小修占一侧，饺子在远处爬 / 玩，妈妈在旁淡淡看（点出「他说归说，家还是在的」）。
- 温情收尾：三人同框，镜头略远，柔光，留白多，不放或只放一句气泡。

## 分镜节奏对应构图

| 格功能 | 构图建议 |
|--------|----------|
| 起：日常铺垫 | 中景，展示家庭环境（客厅 / 卧室），三人自然状态，暖色，留白多。 |
| 承：张力 | 近景特写冲突点（如饺子把饭扣桌上、妈妈举着手机），动作明显。 |
| 转：笑料 | 夸张一点的表情 / 动作（饺子无辜大眼、小修手忙脚乱），可轻微卡通夸张。 |
| 转：歪理 | 小修近景 + 歪头 / 扶眼镜，气泡放歪理；背景虚化或留白。 |
| 合：反转 | 回到饺子身上，用他的动作点破（如递一块积木给爸爸），中景。 |
| 尾：哲理 | 全景柔光，三人同框，安静温暖，气泡少或一句收尾。 |

## 道具池（家庭烟火气）

奶瓶、安抚奶嘴、玩具积木、绘本、婴儿床、推车、餐桌与辅食、拖把、乱丢的拖鞋、手机、相册、脏衣篮、小修的歪眼镜。

用时只选 1-3 个能服务本格意思的，不要堆满。

## 反复刻规则

不要照搬过往单篇的构图与笑点。每篇都从当前主题重新设计分镜与包袱。同一篇内：角色长相、服装、画风必须稳定（依赖 `characters.md` 固定角色块）。同类主题也要换新隐喻与笑点，例如「比较式育儿」可以画饭桌比较，也可以画家长群截图，也可以画玩具比赛——不要每次都一样。
