# 漫画风格 DNA

## 一句话

手绘漫画风、温暖、有笔触与纸感、圆润可爱的家庭育儿绘本漫画。让人想转发给同为父母的朋友，笑着看完还能想一会儿。

像一本给年轻父母看的治愈系「手绘」育儿漫画（indie comic / 水彩手绘感、钢笔线稿），而不是教科书、不是表情包、不是日漫。

## 必须

- 每格 **4:3 横版**单图（本 skill 默认画幅）。
- **手绘漫画质感（核心）**：可见的墨水 / 钢笔笔触，线条略带抖动与不完美（sketchy hand-drawn ink lines, slightly wobbly, not vector-smooth），可有轻微透写 / 草稿感；不是平滑矢量描边、不是干净商业卡通风。
- 柔色填充 + 纸调背景：米白、奶油、浅鹅黄、淡粉、浅蓝、淡绿等低饱和暖色；背景可为带**轻微纸纹 / 暖色调手绘纸感**（允许淡淡纸纹与柔光），但**不要**重噪点、脏污做旧。
- 圆润可爱角色：大头比例、短手短脚、圆脸，萌但不幼稚恶心。
- 每格画面聚焦一个动作 / 一个情绪瞬间，留出放台词气泡的空间。
- 家庭场景真实可感：客厅、卧室、厨房、婴儿床、餐桌、公园——有生活气息与道具细节（奶瓶、玩具、绘本、拖把、鞋）。

## 颜色

- 墨线：深棕 / 近黑的手绘墨线勾边（非纯黑矢量），带笔触轻重。
- 肤色：暖白 / 浅肉色，统一色号，跨格不跳变。
- 饺子：奶白胖脸 + 大眼睛（双眼皮明显）、淡色连体衣或肚兜，整体最「亮」最可爱。
- 小鞋子：知性温柔色系（浅蓝 / 米色 / 淡紫衣裙），发髻 3 个、大眼睛；妆容清淡、漂亮但不艳。无大鞋道具。
- 小修：偏灰 / 偏旧色调（皱旧 T 恤、拖鞋），大脑袋 + 圆框眼镜，头发乱、胡茬、衣领歪；邋遢但眼神清醒。
- 暖色点缀：橙 / 红用于情绪重点、道具；冷色（蓝 / 绿）用于平静、思考或背景。

颜色要「柔」且「手绘」——整体明度偏高、饱和度偏低，读起来像晴天午后的家，笔触可见。

## 绝对不要

- 不要日漫战斗风、赛璐璐重描边、夸张肌肉。
- 不要写实照片、3D 渲染、厚涂油画。
- 不要表情包 / 微信 sticker 风（纯恶搞、崩坏脸、黄暴梗）。
- 不要黑暗、恐怖、惊悚、血腥、过度焦虑的育儿画面。
- 不要说教海报感、政策宣传画、课程封面。
- 不要复杂背景、渐变过猛、阴影过重、重噪点、脏旧做旧。
- 不要把饺子画成学龄儿童（他是 1 岁婴儿：大脑袋大眼、不会说话、靠动作表达）。
- 不要任何「图片由 AI 生成 / AI 生成」类水印文字或角标；生成后若带此类水印必须去除（定位到底部角落文字区域，填成背景纸色）。
- 不要政治、宗教、争议社会议题隐喻；本 skill 只做家庭与育儿的温柔思考。

## 审美方向

要温暖、手绘感、有生活气、让人会心一笑后有点触动。

不要说教、不要油腻段子、不要苦情卖惨、不要过度可爱到发腻、不要干净到像矢量贴纸。
