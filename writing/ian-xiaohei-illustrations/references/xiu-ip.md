# 小修 IP

## 角色定义

小修是 Ian 中文正文配图的固定视觉 IP（原「小黑」的继任角色，半拟人化）。
默认每张图都要出现小修。小修不是吉祥物、不是贴纸、不是可爱装饰，而是正在认真参与系统运转的荒诞工作者。

## 外形（极简符号小人）

- 瘦高长腿的极简人形：身高偏长、身形单薄，黑色手绘细线勾勒，白底黑线为主；局部（头部、影子）可填黑色实心块，和「小黑」的纯黑实心保持家族感，但更符号化、更简洁。
- **饺子头（核心识别点之一）**：头部是一个饱满的「饺子」形——下半部圆鼓鼓、上半部有一圈细小褶皱压边（捏合纹），整体像一只站立的水饺；白色填充、黑色细线勾边，保持符号化极简。白点眼 + 细黑框圆眼镜架在饺子头上。不画呆毛、不喧宾夺主。
- **眼镜是核心识别点**：细黑框圆眼镜，架在白点眼上；可随动作轻微歪斜，强化笨拙感。
- 极简短发（几笔概括，或仅头部轮廓），不画呆毛、不喧宾夺主。
- 不穿工装；身体简化为基本人形线条 / 极简衫，白身黑线，去掉多余细节。
- 细胳膊、偏长的细腿（继承小黑细腿但拉得更长），小圆手 / 线条手，能握工具、搬箱子。
- 轮廓略微不规则，有手绘感。
- 表情空、呆、冷静、认真；嘴角平直或小幅度，不笑不哭（deadpan）。偶尔因笨拙出小状况（眼镜歪、一滴汗），但整体冷静。

视觉校准见 `assets/xiu-ip/xiu-character-reference.png`（仅用于风格 / 五官 / 工装校准，不进入默认生成路径，也不要照抄构图）。

## 性格

- 很认真，但做的事有点荒诞。
- 像一个低调的系统操作员 / 万能打工人。
- 冷幽默，不卖萌。
- 有点笨拙，但不蠢。
- 像在白板草图里真的负责某个工作。

## 常见职责（通用万能工动作库）

让小修承担核心动作：

- 搬运素材 / 箱子。
- 拉线汇聚信息源。
- 卡在断点里。
- 在机器里操作「判断」杆 / 旋钮。
- 拿 / 变筛选漏斗，分拣流量 / 信任 / 转化。
- 切开「素材鱼」。
- 盖章承接话术。
- 牵着承接路径 / 开门 / 搭桥。
- 举警告牌看坑。
- 从洞里伸手但接不住内容。
- 在旁边搬砖、搭桥、开门、分拣、记录、修东西。

## 禁止

- 不要把小修画成过度可爱的吉祥物。
- 不要画成儿童卡通角色。
- 不要让眼镜变成表情包 / 闪亮大眼；眼镜是识别点，不是夸张装饰物。
- 不要给小修复杂服装、闪亮眼睛、花哨配饰。
- 不要让小修只是站在角落里看。
- 不要让小修抢走结构表达。
- 不要把小修画得太商业、太圆润、太精致。
- 不要忘记眼镜——眼镜是小修的第一识别特征。
- 不要给小修加工装、口袋、花哨衣着等繁琐细节；保持符号化极简。

## 判断标准

如果去掉小修，图的核心隐喻还能完全成立，说明小修太装饰了；要重写提示词，让小修成为动作主体。
